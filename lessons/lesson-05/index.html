Перебираючі методи масивів

Example 2 — Arrow function

Перепиши функцію у стрілкову.

 function sum(a, b) { return a + b; }

Що зробити:

  • зробити arrow function

Example 3 — map

Збільш кожне число в 2 рази.

 const numbers3 = [1, 2, 3, 4];

Що зробити:

 [2, 4, 6, 8]

Example 4 — map (реальний кейс)

Отримай список імен користувачів.

 const users4 = [ { name: "Olena" }, { name: "Ivan" }, { name: "Anna" } ];

Результат:

 ["Olena", "Ivan", "Anna"]

Example 5 — flatMap

Розгорни вкладені масиви.

 const arr = [1, 2, [3, 4], [5, 6]];

Що зробити:

 [1, 2, 3, 4, 5, 6]

Example 6 — every

Перевір, чи всі числа більше 10.

 const numbers6 = [12, 15, 18, 20];

Результат:

 true

Example 7 — some

Перевір, чи є хоча б одне число більше 100.

 const numbers7 = [10, 20, 30, 150];

Результат:

 true

Example 8 — reduce (сума)

Порахуй суму всіх чисел.

 const numbers8 = [1, 2, 3, 4];

Результат:

 10

Example 9 — reduce (реальний кейс)

Порахуй загальну ціну кошика.

 const cart = [ { price: 100 }, { price: 200 }, { price: 300 } ];

Результат:

 600

Example 10 — toSorted

Відсортуй числа по зростанню без зміни оригінального масиву.

 const numbers10 = [5, 2, 9, 1];

Результат:

 [1, 2, 5, 9]

Умова: використовувати toSorted()

Example 11 — toSorted (об'єкти)

Відсортуй користувачів за віком.

 const users11  = [ { name: "Olena", age: 25 }, { name: "Ivan", age: 20 }, { name: "Anna", age: 30 } ];

Результат:

 20 → 25 → 30

Example 12 — callback + map

Створи функцію, яка приймає масив і callback.

 function transform(arr, callback) {}

Що зробити:

  • застосувати callback до кожного елемента
  • повернути новий масив

Приклад:

 transform([1,2,3], x => x * 2) → [2,4,6]
