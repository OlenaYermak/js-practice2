Об'єкти

Завдання 1 — Доступ до властивостей

Отримай значення з об’єкта.

 const user = { name: "Olena", age: 25, isOnline: true };

Що зробити:

Виведи в консоль:

 "Name: Olena" "Age: 25" "Online: true"

Завдання 2 — Додавання властивості

Додай нове поле.

 const user1 = { name: "Olena" };

Що зробити:

  • додай поле email зі значенням "olena@gmail.com"

Вивести:

 { name: "Olena", email: "olena@gmail.com" }

Завдання 3 — Зміна значення

Онови дані користувача.

 const user2 = { name: "Olena", age: 25 };

Що зробити:

  • змінити age на 26

Завдання 6 — Перебір ключів

Виведи всі ключі об’єкта.

 const user6 = { name: "Olena", age: 25, city: "Kyiv" };

Результат:

 name age city

Умова: використати for...in

Завдання 7 — Перебір значень

Виведи всі значення.

 const user7 = { name: "Olena", age: 25, city: "Kyiv" };

Результат:

 Olena 25 Kyiv

Завдання 8 — Ключ + значення

Зроби форматований вивід.

 const user8 = { name: "Olena", age: 25 };

Результат:

 name: Olena age: 25

Завдання 9 — Підрахунок властивостей

Порахуй кількість полів.

 const user9 = { name: "Olena", age: 25, city: "Kyiv" };

Результат:

 3

Завдання 11 — Дані користувача

Відфільтруй дані.

 const user = { name: "Olena", age: 25, password: "12345", email: "olena@gmail.com" };

Що зробити:

  • вивести всі поля, КРІМ password

Завдання 12 — Сума значень

Порахуй суму чисел в об’єкті.

 const salaries = { John: 1000, Anna: 1500, Mike: 800 };

Результат:

 3300

Завдання 13 — Найбільше значення

Знайди максимальну зарплату.

 const salaries = { John: 1000, Anna: 1500, Mike: 800 };

Результат:

 1500

Завдання 14 — Оновлення об’єкта

Зміни значення.

 const product = { name: "Laptop", price: 1000 };

Що зробити:

  • якщо price > 500 → зменшити на 10%

Завдання 15 — Копія об’єкта

Створи копію.

 const user = { name: "Olena", age: 25 };

Що зробити:

  • створити новий об’єкт newUser
  • додати поле isAdmin: false
