Об'єкти

Завдання 1 — Доступ до властивостей

Отримай значення з об’єкта.

 const user = { name: "Olena", age: 25, isOnline: true };

Що зробити:

Виведи в консоль:

 "Name: Olena" "Age: 25" "Online: true"

Завдання 2 — Додавання властивості

Додай нове поле.

 const user1 = { name: "Olena" };

Що зробити:

  • додай поле email зі значенням "olena@gmail.com"

Вивести:

 { name: "Olena", email: "olena@gmail.com" }

Завдання 3 — Зміна значення

Онови дані користувача.

 const user2 = { name: "Olena", age: 25 };

Що зробити:

  • змінити age на 26

Завдання 4 — Перевірка властивості

Перевір, чи є поле.

 const user4 = { name: "Olena", age: 25 };

Що зробити:

  • якщо є email → "Email exists"
  • інакше → "No email"

Завдання 5 — Видалення властивості

Видали поле з об’єкта.

 const user5 = { name: "Olena", age: 25, password: "12345" };

Що зробити:

  • видалити поле password

Завдання 6 — Перебір ключів

Виведи всі ключі об’єкта.

 const user6 = { name: "Olena", age: 25, city: "Kyiv" };

Результат:

 name age city

Умова: використати for...in

Завдання 7 — Перебір значень

Виведи всі значення.

 const user7 = { name: "Olena", age: 25, city: "Kyiv" };

Результат:

 Olena 25 Kyiv

Завдання 8 — Ключ + значення

Зроби форматований вивід.

 const user8 = { name: "Olena", age: 25 };

Результат:

 name: Olena age: 25

Завдання 9 — Підрахунок властивостей

Порахуй кількість полів.

 const user9 = { name: "Olena", age: 25, city: "Kyiv" };

Результат:

 3

Завдання 10 — Пошук значення

Перевір, чи є значення в об’єкті.

 const user10 = { name: "Olena", age: 25, city: "Kyiv" };

Що зробити:

  • якщо є значення "Kyiv" → "Found"
  • інакше → "Not found"

Завдання 12 — Сума значень

Порахуй суму чисел в об’єкті.

 const salaries = { John: 1000, Anna: 1500, Mike: 800 };

Результат:

 3300

Знайди максимальну зарплату.

 const salaries2 = { John: 1000, Anna: 1500, Mike: 800 };

 1500

  • якщо price > 500 → зменшити на 10%

Завдання 16 — Банківський акаунт (об’єкти + масиви + методи)

Реалізуй об’єкт акаунта з історією транзакцій.

🔹 Вхідні дані:

 const Transaction = { DEPOSIT: "deposit", WITHDRAW: "withdraw", }; const account = { balance: 0, transactions: [], };

🔹 Що зробити:

Реалізувати наступні методи в об’єкті account 👇

1. createTransaction(amount, type)

Що робить:

  • створює об’єкт транзакції
  • об’єкт має вигляд:
 { id: унікальний_id, type: "deposit" або "withdraw", amount: число }

Повертає: об’єкт транзакції

2. deposit(amount)

Що робить:

  • збільшує balance
  • створює транзакцію через createTransaction
  • додає її в transactions

Результат:

 balance збільшено транзакція додана в історію

3. withdraw(amount)

Що робить:

  • зменшує balance
  • створює транзакцію
  • додає її в історію

Умова:

  • якщо amount > balance →
 "Недостатньо коштів"

4. getBalance()

Що зробити:

  • повернути поточний баланс

5. getTransactionDetails(id)

Що зробити:

  • знайти транзакцію по id
  • повернути об’єкт транзакції

6. getTransactionTotal(type)

Що зробити:

  • порахувати суму всіх транзакцій певного типу

Приклад:

 deposit → сума всіх поповнень withdraw → сума всіх зняттів

🔹 Приклад використання:

 account.deposit(1000); account.deposit(500); account.withdraw(300); console.log(account.getBalance());  console.log(account.getTransactionTotal("deposit"));  console.log(account.getTransactionTotal("withdraw"));

Example 21 — Rest (збір властивостей)

Витягни name, а всі інші властивості збережи в окремий об'єкт через ...

 const user21 = { name: "Olena", age: 25, city: "Kyiv", };
👉 Очікується: 
 name = "Olena" rest = { age: 25, city: "Kyiv" }
👉 Вивести: 
 console.log(name) console.log(rest)

Example 24 — Комбінований (spread + метод)

Створи метод addProperty(key, value), який повертає новий об'єкт з доданою властивістю (не мутуючи оригінал).

 const user24 = { name: "Olena", };

👉 Очікується:

 console.log(updatedUser) console.log(user) // не змінений
