ООП. Класи

Example 1 — Контекст виконання (this)

Створи метод об'єкта, який використовує this.

 const user = { name: "Olena", showName() { } };

Що зробити:

  • метод має вивести ім'я користувача через this.name

Очікується:

 Olena

Example 2 — Втрата контексту

Виправ помилку втрати this.

 const user2 = { name: "Olena", showName() { console.log(this.name); } }; const fn = user2.showName; fn();

Що зробити:

  • зробити так, щоб в консоль вивелось:
 Olena

Умова: використати bind()

Example 3 — call()

Виклич функцію з іншим контекстом.

 function showCity() { console.log(this.city); } const user = { city: "Kyiv" };

Що зробити:

  • викликати функцію так, щоб вивелось:
 Kyiv

Умова: використати call()

Example 5 — Prototype

Додай метод через prototype.

 function User(name) { this.name = name; }

Що зробити:

  • додати метод sayHello() через prototype

Очікується:

 Hello Olena

Example 6 — Перевірка prototype

Перевір, чи існує метод у prototype.

 function Car(model) { this.model = model; } Car.prototype.showModel = function() { console.log(this.model); };

Що зробити:

  • створити об'єкт
  • викликати метод

Example 7 — Основи OOP

Створи клас користувача.

 class User2 { }

Що зробити:

  • додати constructor
  • зберігати name і age

Приклад:

 const user7 = new User2("Olena", 25);

Example 8 — Метод класу

Додай метод до класу.

 class User3 { constructor(name) { this.name = name; } }

Що зробити:

  • додати метод getName()

Очікується:

 Olena
