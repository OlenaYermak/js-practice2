Масиви та функції.

Завдання 1 — Сума елементів масиву

Обчисли суму всіх чисел.

 const numbers = [1, 2, 3, 4, 5];

Що зробити:

 Використовуючи цикл, порахуй суму і виведи

Завдання 2 — Парні числа з масиву

Знайди тільки парні числа.

 const numbers2 = [1, 223, 30, 4, 5, 6, 8, 22, 15, 90, 60, 53];

Що зробити:

Створи новий масив з парними числами

             [30, 4, 6, 8, 22, 90, 60]

Умова: використати for

Завдання 3 — Найбільше число

Знайди максимум.

 const numbers3 = [5, 12, 3, 20, 8];

Що зробити:

Вивести:

 20

Завдання 4 — Реверс масиву

Переверни масив без методу reverse().

 const numbers4 = [1, 2, 3];

Результат:

 [3, 2, 1]

Завдання 5 — Підрахунок елементів

Порахуй кількість елементів більших за 10.

 const numbers5 = [5, 15, 8, 20, 3];

Результат:

 2

Завдання 6 — includes

Перевір наявність елемента.

 const fruits = ["apple", "banana", "orange"];

Що зробити:

Перевір, чи є "banana"

Вивести:

 "Found"

Завдання 7 — join

Об’єднай елементи.

 const words = ["Hello", "world"];

Результат:

 "Hello world"

Завдання 8 — split

Розбий рядок на масив.

 const text = "a,b,c";

Результат:

 ["a", "b", "c"]

Завдання 9 — Function Declaration

Напиши функцію для суми.

 function sum(a, b) { }

Що зробити:

Повернути суму двох чисел

Завдання 10 — Function Expression

Створи функцію перевірки парного числа.

 const isEven = function(num) { };

Результат:

повертає true або false

Завдання 11 — Параметри за замовчуванням

Задай значення за замовчуванням.

 function greet(name) { }

Що зробити:

  • якщо name не передано → "Guest"

Результат:

 "Hello, Guest"

Завдання 12 — arguments (псевдомасив)

Порахуй суму всіх аргументів.

 function sumAll() { }

Приклад виклику:

 sumAll(1, 2, 3, 4);

Результат:

 10

Завдання 13 — Ітерація по масиву у функції

Порахуй суму масиву.

 function getTotal(numbers) { }

Результат:

 getTotal([1,2,3]) → 6

Завдання 14 — універсальна функція

Зроби функцію, яка приймає будь-яку кількість чисел.

 function getAverage() { }

Що зробити:

  • використати arguments
  • повернути середнє значення
