Розгалудження та цикли

Завдання 1 — Доступ до акаунта

Перевір, чи користувач може отримати доступ до системи.

 const isLoggedIn = true;
 const isEmailVerified = false;

Умова:

  • Користувач має бути залогінений
  • Email має бути підтверджений

Що зробити:

Виведи в консоль:

  • "Access granted" — якщо обидві умови виконані
  • "Access denied" — якщо ні

Завдання 2 — Знижка

Перевір, чи застосовується знижка.

 const isNewUser = false; 
 const hasPromo = true;

Умова:

  • Знижка дається, якщо користувач новий АБО має промокод

Що зробити:

Виведи в консоль:

  • "Discount applied"
  • "No discount"

Завдання 3 — Перевірка пароля

Оціни безпеку пароля.

 const password = "qwerty123";

Умова:

  • Менше 8 символів → слабкий
  • Містить "123" → небезпечний
  • Інакше → нормальний

Що зробити:

Виведи в консоль один із варіантів:

  • "Weak password"
  • "Unsafe password"
  • "Password is valid"

Завдання 4 — Форматування імені

Приведи ім’я до правильного формату.

         const name = " olena ";

Що зробити:

  • Прибери пробіли з початку і кінця
  • Зроби першу літеру великою

Результат:

 "Olena"

Дія: створи нову змінну formattedName і виведи її в консоль

Завдання 5 — Доставка

Визнач вартість доставки.

             const orderAmount = 800;

Умова:

  • Якщо сума ≥ 1000 → безкоштовно
  • Інакше → 100 грн

Що зробити:

Виведи в консоль:

  • "Free delivery"
  • "Delivery cost: 100 UAH"

Завдання 6 — Перевірка email

Перевір валідність email.

             const email = "user@gmail.com";

Умова:

  • Має містити @
  • Має закінчуватись на gmail.com

Що зробити:

Виведи в консоль:

  • "Valid email"
  • "Invalid email"

Завдання 7 — Модерація коментаря

Перевір текст коментаря.

 const comment = "This is stupid";

Умова:

  • Якщо містить "stupid" або "hate" → блок

Що зробити:

Виведи:

  • "Comment blocked"
  • "Comment allowed"

Завдання 8 — Доступ за віком

Перевір доступ користувача.

 const age = 17; 
const hasID = true;

Умова:

  • 18+ → повний доступ
  • 16–17 + ID → обмежений
  • Інакше → відмова

Що зробити:

Виведи:

  • "Access granted"
  • "Limited access"
  • "Access denied"

Завдання 9 — Генерація slug

Зроби URL-friendly рядок.

 const title = " JavaScript for Beginners ";

Що зробити:

  • Прибрати пробіли
  • Перевести в нижній регістр
  • Замінити пробіли на -

Результат:

 "javascript-for-beginners"

Дія: створи змінну slug і виведи її

Завдання 10 — Фінальна ціна

Розрахуй ціну з урахуванням знижок.

 const price = 200; 
 const discount = 10; 
const isMember = true;

Умова:

  • Якщо isMember → -10%
  • Якщо discount > 0 → ще мінус discount%

Що зробити:

Виведи:

  • "Final price: ___"

Завдання 11 — День тижня

Визнач тип дня.

 const day = "Saturday";

Що зробити:

  • "Monday"–"Friday" → "Working day"
  • "Saturday", "Sunday" → "Weekend"
  • default → "Invalid day"

Дія: використай switch і виведи в консоль

Завдання 12 — Мова сайту

Показати привітання залежно від мови.

 const lang = "ua";

Що зробити:

  • "ua" → "Привіт"
  • "en" → "Hello"
  • "pl" → "Cześć"
  • default → "Unknown language"

Завдання 13 — Очищення тексту

Приведи рядок до нормального вигляду.

 const text = " HELLO world ";

Що зробити:

  • прибрати пробіли
  • зробити: "Hello world"

Дія: створи змінну result і виведи її

Завдання 14 — Перевірка домену

Перевір сайт.

 const url = "https://my-site.com";

Що зробити:

  • якщо починається з https → "Secure"
  • інакше → "Not secure"

Завдання 15 — Вирізання username

Отримай ім’я користувача з email.

 const email = "olena@gmail.com";

Результат:

 "olena"

Підказка: split()

Завдання 16 — Цензура

Замаскуй погані слова.

 const text = "This is stupid and bad";

Що зробити:

  • замінити "stupid" → "***"
  • замінити "bad" → "***"

Завдання 16 — Ініціали

Зроби ініціали з імені.

 const fullName = "Olena Yermak";

Результат:

 "O.Y."

Завдання 17 — Логін система

Перевір доступ користувача.

 const username = " admin "; const password = "Admin123";

Що зробити:

  • очистити username
  • якщо username === "admin" і пароль містить число →

Вивести:

  • "Welcome admin"
  • "Access denied"

Завдання 18 — Категорія товару

Визнач знижку.

 const category = "electronics"; const price = 1000;

Умова:

  • "electronics" → -10%
  • "clothes" → -20%
  • "food" → без знижки

Що зробити:

Використай switch і виведи:

 "Final price: ___"

Завдання 19 — Повідомлення користувача

Згенеруй повідомлення.

 const name = " olena "; const isLoggedIn = true;

Що зробити:

  • очистити ім’я
  • зробити "Olena"

Вивести:

  • якщо залогінений → "Hello, Olena"
  • інакше → "Please log in"

Завдання 20 — Перевірка файлу

Перевір формат файлу.

 const fileName = "photo.png";

Що зробити:

  • якщо закінчується на ".jpg" або ".png" → "Image file"
  • якщо ".pdf" → "Document"
  • інакше → "Unknown format"

Завдання 21 — Генерація пароля

Перевір і покращ пароль.

 const password = "qwerty";

Що зробити:

  • якщо < 6 символів → додати "123"
  • зробити всі букви великими

Результат:

 "QWERTY123"

Завдання 22 — Кошик товарів

Розрахуй суму.

 const itemName = "Laptop"; const price = 500; const quantity = 2;

Що зробити:

  • порахувати суму
  • зробити назву маленькими буквами

Вивести:

 "laptop x2 = 1000"
