Завдання 1:

Якщо дано невід'ємне ціле число, наприклад, 3, поверніть рядок із бурмотінням: «1 вівця...2 вівці...3 вівці...». Вхідні дані завжди будуть коректними, тобто жодних від'ємних цілих чисел.

Завдання 4:

Створіть функцію, яка приймає ціле число як аргумент і повертає "Even" для парних чисел або "Odd" для непарних чисел.

Завдання 5:

Завдання 6:

Ваші однокласники попросили вас скопіювати для них деякі документи. Ви знаєте, що однокласників є 'n', і документи мають 'm' сторінок. Ваше завдання — підрахувати, скільки порожніх сторінок вам потрібно. Якщо n < 0 або m < 0, поверніть 0. Приклад: n=5, m=5: 25 n=-5, m=5: 0

Завдання 7:

Ваше завдання — створити функцію, яка може приймати будь-яке невід'ємне ціле число як аргумент і повертати його з цифрами в порядку спадання. По суті, переставте цифри, щоб отримати найбільше можливе число.

Завдання 8:

Банкомати дозволяють використовувати 4- або 6-значні PIN-коди, і PIN-коди не можуть містити нічого, окрім рівно 4 або рівно 6 цифр. Якщо функції передається дійсний рядок PIN-коду, повертається значення true, інакше повертається значення false.

Завдання 9:

Напишіть функцію, яка обчислює середнє значення чисел у заданому масиві. Примітка: Порожні масиви повинні повертати 0.

Завдання 10:

Напишіть функцію, яка видаляє пробіли з рядка, а потім повертає результуючий рядок. Приклади (Вхід -> Вихід): "8 j 8 mBliB8g imjB8B8 jl B" -> "8j8mBliB8gimjB8B8jlB" "8 8 Bi fk8h B 8 BB8B B B B888 c hl8 BhB fd" -> "88Bifk8hB8BB8BBBB888chl8BhBfd" "8aaaaa dddd r " -> "8aaaaaddddr"

Завдання 11:

Тролі атакують ваш розділ коментарів! Поширений спосіб вирішення цієї ситуації — видалити всі голосні літери з коментарів тролів, нейтралізуючи загрозу. Ваше завдання — написати функцію, яка приймає рядок і повертає новий рядок з усіма видаленими голосними. Наприклад, рядок «This website is for losers LOL!» перетвориться на «Ths wbst s fr lsrs LL!». Примітка: в цьому завданні "y" не вважається голосною.
